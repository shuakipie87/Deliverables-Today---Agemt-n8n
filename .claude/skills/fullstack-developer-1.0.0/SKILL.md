# Full-Stack Developer Assistant Skill

**Transform ClawdBot into your expert full-stack development partner**

## 🎯 Overview

This skill enhances ClawdBot with comprehensive full-stack development capabilities, making it your go-to assistant for:

- **Frontend Development** (React, Vue, Angular, Next.js, etc.)
- **Backend Development** (Node.js, Python, Go, PHP, etc.)
- **Database Management** (PostgreSQL, MongoDB, MySQL, Redis)
- **API Development** (REST, GraphQL, WebSocket)
- **DevOps & CI/CD** (Docker, Kubernetes, GitHub Actions)
- **Testing & QA** (Jest, Pytest, Cypress, etc.)
- **Code Quality** (Linting, formatting, best practices)

## 💡 Usage Examples

### Command: `/dev`
General development assistance

**Example:**
```
/dev Create a React component for user authentication
```

### Command: `/code`
Code generation and fixes

**Example:**
```
/code Fix the memory leak in this useEffect hook
```

### Command: `/debug`
Debug assistance

**Example:**
```
/debug Why is my API returning 500 errors?
```

### Command: `/deploy`
Deployment assistance

**Example:**
